# Line Chart

**\[experimental\]**

A high-level
[`plotly::plot_ly()`](https://rdrr.io/pkg/plotly/man/plot_ly.html)-function
for charting Open, High, Low and Close prices.

## Usage

``` r
pline(price = "close", ...)
```

## Arguments

- price:

  A [character](https://rdrr.io/r/base/character.html)-vector of
  [length](https://rdrr.io/r/base/length.html) 1. "close" by default.

- ...:

  For internal use. Please ignore.

## Value

An [invisible](https://rdrr.io/r/base/invisible.html)
[`plotly::plot_ly()`](https://rdrr.io/pkg/plotly/man/plot_ly.html)-object.

## See also

Other price charts:
[`chart()`](https://serkor1.github.io/cryptoQuotes/reference/chart.md),
[`kline()`](https://serkor1.github.io/cryptoQuotes/reference/kline.md),
[`ohlc()`](https://serkor1.github.io/cryptoQuotes/reference/ohlc.md)
